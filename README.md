# Mailing Tools

This project is a web-based collection of utilities designed to simplify mailing-related configurations and string generation tasks.  
It provides an easy-to-use interface with multiple tools for developers, email marketers, and system admins.

- **Dynamic Hex Block & Code Generator**  
  Generate dynamic hexadecimal blocks and codes for technical configurations.

- **Encoder & Decoder**  
  Encode and decode strings for various formats (Base64, URL encoding, etc.).
